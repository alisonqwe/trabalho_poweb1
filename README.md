Objetivo do sistema

O sistema Minha Estante Digital tem como objetivo permitir que o usuário organize filmes e séries em uma lista pessoal, podendo cadastrar, visualizar, filtrar, excluir e acompanhar estatísticas das mídias cadastradas.

Funcionalidades principais

Você vai fazer estas:

Cadastrar filme ou série.
Mostrar os dados em cards.
Filtrar por título.
Filtrar por tipo: filme ou série.
Filtrar por status.
Excluir item cadastrado.
Mostrar estatísticas.
Salvar os dados no navegador.


NA PARTE DO HTML :
O HTML foi usado para estruturar a página. O Bootstrap foi usado como framework CSS para facilitar a criação de uma interface responsiva.

O JavaScript foi separado em um arquivo externo chamado script.js. Ele é responsável por cadastrar as mídias, atualizar os cards, aplicar filtros, atualizar as estatísticas e salvar os dados no navegador usando localStorage.
